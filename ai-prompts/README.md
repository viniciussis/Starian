# AI-Assisted Development Prompts

Este diretório contém os scripts, diretrizes e prompts que foram arquitetados por min (Desenvolvedor) para auxiliar modelos de inteligência artificial (como Gemini) a gerarem partes isoladas do projeto, revisarem o código e gerarem massa de testes.

A Starian é uma empresa **AI First**, portanto, o uso de inteligência artificial durante este desafio técnico foi feito de forma metodológica, estrutural e não como "código cego". 

### Estrutura Planejada

- **`.cursorrules`** (Raiz): Arquivo onde paramatrizei as regras de Clean Code (NestJS/Vue3) para o agente na IDE.
- **`generate-crud.md`** (Brevemente): Prompts de automação de scaffolding que garantem o uso de DTOs nativos de infra.
- **`code-review-checklist.md`** (Brevemente): Instrução pré-configurada para que a IA realize análise estática em busca de melhorias de performance.

Esta forma de trabalhar demonstra domínio não apenas da sintaxe (Stack: Vue 3 Single-SPA & NestJS), mas capacidade arquitetural e orquestração de Agentes IA para aumentar produtividade e manter alto padrão na entrega.
